Finance Tracker - Android Application (WIP 🚧)
Bu proje, kullanıcıların günlük gelir ve giderlerini takip edebilmeleri amacıyla Kotlin kullanılarak geliştirilmektedir. Proje şu an geliştirme aşamasındadır (Work In Progress).

Mevcut Özellikler
Kullanıcı Arayüzü: Modern Android tasarım prensiplerine uygun arayüz çalışmaları.

Veri Yönetimi: Gelir ve gider kalemlerinin listelenmesi ve kategorize edilmesi.

Teknolojiler: Kotlin, Android Studio, XML.

Gelecek Planları (Geliştirilme Süreci)
Yerel Veritabanı: SQLite veya Room entegrasyonu ile verilerin kalıcı saklanması.

Görselleştirme: Harcamaların pasta grafiği (Pie Chart) ile analiz edilmesi.

Firebase: Bulut tabanlı senkronizasyon ve kullanıcı girişi.
